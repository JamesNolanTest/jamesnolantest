1: How to Run

Script is built using Python 2.7
It utilises 2 additional libraries 'Selenium' & 'Github'
If these are not already installed on your machine they can be pulled using pip.

pip install selenium
pip install PyGithub

Double clicking the script will run the tests.
Steps print out their actions for logging purposes.

Issues are generated in the repo https://github.com/JamesNolanTest/jamesnolantest
